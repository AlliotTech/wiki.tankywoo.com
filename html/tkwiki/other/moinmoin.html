安装篇

参考官方的HowTo

虽然我用的是Gentoo，但是基本都大同小异，我参考的是Ubuntu的安装，非常详细

使用virtualenv构造一个隔离的python环境

官方用了virtualenv，搭建一个隔离的环境 

# Install and activate python virtualenv
emerge -av  virtualenv
mkdir -p /var/www/moin
virtualenv /var/www/moin/pythonenv
source /var/www/moin/pythonenv/bin/activate

# Download and install moinmoin:
cd /tmp
wget http://static.moinmo.in/files/moin-1.9.6.tar.gz
tar zxvf moin-1.9.6.tar.gz
python setup.py install	# 这样就会将一个python包安装在/var/www/moin/pythonenv下

# Deactivate python virtualenv
deactivate

# Copy wiki to /var/www/moin
cp -r ./wiki /var/www/moin/

# Copy configs to wiki root directory
cd /var/www/moin/wiki/
cp config/wikiconfig.py ./
cp server/moin.wsgi ./

做基本的配置

vim /srv/www/moin/wiki/moin.wsgi

sys.path.insert(0, '/var/www/moin/pythonenv/lib/python2.7/site-packages/')
sys.path.insert(0, '/var/www/moin/wiki/')

#Fix permission
chown nginx:nginx -R /var/www/moin
chmod o-rwx -R /var/www/moin

部署uwsgi

# Add an uwsgi.xml config file:
vim /var/www/moin/wiki/uwsgi.xml

<uwsgi>
    <uid>nginx</uid>
    <gid>nginx</gid>
    <plugin>python27</plugin>
    <socket>/var/www/moin/moin.sock</socket>
    <wsgi-file>/var/www/moin/wiki/moin.wsgi</wsgi-file>
    <limit-as>256</limit-as>
    <processes>8</processes>
    <memory-report/>
    <vhost/>
    <no-site/>
</uwsgi>

然后启动uwsgi 

uwsgi -x /var/www/moin/wiki/uwsgi.xml

最后配置Nginx

贴一下本机的Nginx配置 

# MonMoin
server {
    listen 10086;
    server_name localhost;

    access_log /var/log/nginx/wiki.access_log main;
    error_log /var/log/nginx/wiki.error_log info;

    location ^~ /wiki {
        include uwsgi_params;
        uwsgi_pass unix:///tmp/wiki.sock;
        uwsgi_param UWSGI_PYHOME /var/www/moinmoin/python-env/;
        uwsgi_param UWSGI_CHDIR /var/www/moinmoin/wiki/;
        uwsgi_param UWSGI_SCRIPT moin_wsgi;
        uwsgi_param SCRIPT_NAME /wiki;
        uwsgi_modifier1 30;
    }

    location ^~ /wiki/moin_static196/ {
        alias /var/www/moinmoin/python-env/lib/python2.7/site-packages/MoinMoin/web/static/htdocs/;
    }
}

重启Nginx，打开localhost:10086，基本就OK了
