wiki文件中代码高亮

使用nested_syntaxes选项，比如我的.vimrc文件: 

'nested_syntaxes' : {'python': 'python', 'c++': 'cpp', 'shell': 'sh'}}]

里面保存的是键值对，键是brush调用时的name，值是vim filetype
然后在{{{code}}}里就可以按filetype来高亮代码了, Example: 

XXXclass="brush: c++"
#include "helloworld.h"
int helloworld()
{
	printf("hello world");
}
XXX

注：XXX代表花括号

更多可以help vimwiki，然后搜vimwiki-option-nested_syntaxes

More：如果考虑在html时对<pre>代码高亮，可以考虑syntaxhighlight插件
