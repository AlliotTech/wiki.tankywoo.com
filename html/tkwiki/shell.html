Linux Shell

什么是Shell

shell是一个作为用户与Linux系统间接口的程序，它允许用户向操作系统输入需要执行的命令

基本

重定向

0	标准输入 <  <<
1	标准输出 >  >>
2	错误输出 2> 2>>

<是输出， <<是以附加的方式输出
>& kill -9 1234 >killouterr.txt 2>&1

管道

管道用操作符 '|'

脚本

格式

第一行指定执行shell的文件，一般是#!/bin/bash或#!/bin/sh

运行

* 直接运行 sh xx.sh 或者 bash xx.sh
* 可执行文件 先chmod +x xx.sh 然后 ./xx.sh
